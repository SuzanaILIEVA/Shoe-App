# Shoe-App 
You can search for your favorite shoe by size, color, gender and price and view its details. It is designed responsively for every screen.


## Technologies
- React
- Typescript

## Libraries
- react-router-dom
- tailwind
- axios
- json-server
- @tanstack/react-query
- react-toastify
- @uidotdev/usehooks
